# Client

The frontend of this application.

### Prerequisites

#### Dependencies

* Node.js
* NPM
* React
  * react-dom
  * react-scripts
  * react-router-dom
* query-string

### Installing

```
cd react-nodejs-passport-app
npm install
```

## Getting Started

```
npm start
```

Visit [http://localhost:3000](http://localhost:3000)
